# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |


## Reporting a Vulnerability

Thank you for contributing to the security of this project. Please report any
security vulnerabilities using [Github's report
function.](https://github.com/yongjae354/python-rekor-monitor/security/advisories/new)

When a report is recieved and the vulnerability is confirmed, a fix will be
issued in the form of a new release, as well as a report that explains the
vulnerability.
